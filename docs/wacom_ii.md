# Wacom II-S:

## Usage

- Follow [General Usage](./general_usage.md)

- Run `KuuubeTD/wacom_ii.py`

## Notes

### Settings Commands Applied by Driver:

```
RE
PH1
AS1
IN0
SR
BA8
```

### Setting Data

`RE` reset to default command set

`PH1` pressure mode: on

`AS1` data format: binary

`IN0` increment mode: 0

`SR` stream mode: on

`BA8` baud rate: 19200
